## 同格性の設計判断（交絡の遮断）

1. **A2 の環境中立化**：公開版 SKILL.md には Claude Code / Gemini / Google AI Studio への言及があり、A5（中立化済み）とそのまま比較すると「語彙」以外の差が混入する。A2 は A5 と**同一の中立化**（frontmatter除去・由来を「原型となる編成の移植」化・環境節の一般化）を施し、**A2 と A5 の差分が語彙のみ**になるよう揃えた。仏教語彙・機構・条項は公開版と同一。
2. **安全条項の共有**：危機時条項（相談先の電話番号を含む）と「役割名（部名）は権限ではない」は A2・A3・A5 の三腕すべてに同一機能で置いた。安全は統制対象ではないため、腕間の差分から除外されている。
3. **A3 の禁止語検収**：確度・確信・検証・検分・検算・訂正・修正・根拠・出典・帰属・倫理・利益・確認・真偽・誤り 等が0件であることを、執筆エージェントの自己チェックとは**独立に** grep で確認済み（2026-07-10）。
4. **A5 の残存語検収**：仏教語彙30語（如来・菩提・衆生・大円鏡・覚醒・荘厳・儀式・顕現ほか）が0件であることを独立に grep で確認済み。機能条項（用量制御・出口・危機・統合達成判定の硬化・事後検分a〜d・ブリッジング・倫理規定全項）の残存も確認済み。
5. **既知の残差**：A2・A3・A5 の字数は完全一致ではない（5,900〜7,500）。長さの効果は A3（構造・長さ）対 A1 の対比で主に吸収し、字数の微差は限界として記録する。
